= taskomaly

access your Tasko papers. in ruby.

== poof

 t = Taskomaly::With :user => 9999, :key => 'd9cca721a735dac4efe709e0f3518373' 

== or

create this file:

 # ~/.tasks.yml
 user: 9999
 key: d9cca721a735dac4efe709e0f3518373

and do this:

 t = Taskomaly::From '~/.tasks.yml'

= now what

== getting your papers

 p = t.request :papers # ['Paper One', 'Paper Two']
